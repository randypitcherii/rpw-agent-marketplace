# Decisions

Architecture Decision Records (ADRs) and durable decisions.

## Conventions

- One decision per file.
- Include: context, decision, consequences.
- Optional: alternatives considered, status (accepted/deprecated/superseded).

## Format

```markdown
# ADR-XXX: Title

**Status:** Accepted
**Date:** YYYY-MM-DD

## Context
...

## Decision
...

## Consequences
...
```
